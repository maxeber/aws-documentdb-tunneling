
# AWS VPC Tunneling Clients

[![lerna](https://img.shields.io/badge/maintained%20with-lerna-cc00ff.svg)](https://lernajs.io/)

Client wrapper wich allow SSH tunneling into a VPC by going through a EC2 instance hosted in the same VPC while also managing connection to your `local`/`dev` environment:

- [mongodb](https://github.com/maxeber/aws-tunneling/tree/master/packages/mongodb)
- [mongoose](https://github.com/maxeber/aws-tunneling/tree/master/packages/mongoose)
- [elasticsearch](https://github.com/maxeber/aws-tunneling/tree/master/packages/elasticsearch)
